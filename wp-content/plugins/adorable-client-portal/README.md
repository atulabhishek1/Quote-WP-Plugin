# Adorable Client Portal

Premium Interior Design CRM & Client Portal for WordPress.

## Description

A powerful client portal plugin designed for interior designers to manage clients, projects, invoices, and communications — all within WordPress.

## Requirements

- WordPress 6.0+
- PHP 8.0+

## Installation

1. Upload the `adorable-client-portal` folder to `/wp-content/plugins/`
2. Activate the plugin through the **Plugins** menu in WordPress
3. Navigate to **Client Portal** in the admin menu to configure

## Development

### Setup

```bash
composer install
```

### Code Standards

```bash
composer run phpcs
```

### Tests

```bash
composer run test
```
