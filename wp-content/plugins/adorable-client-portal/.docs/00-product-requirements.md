# Product Requirements

## Vision
Build a premium WordPress CRM and Client Portal for Adorable Creations.

## Primary Modules
- Dashboard
- Clients
- Projects
- Quotes
- Payments
- Gallery
- Documents
- Progress Tracker
- Reports
- Settings

## Core Workflow
Lead → Client → Project → Quote → Approval → Payment → Production → Installation → Handover

## Functional Requirements
- Modern responsive UI
- Role based access
- Shareable quote links
- PDF generation
- Razorpay integration
- Activity logs
- Search, filters and reports

## Non-functional
- PHP 8+
- WordPress coding standards
- OOP
- Secure
- Scalable
- Mobile friendly
